Privacy Policy

Last Updated: May 16, 2024

This Privacy Policy explains how Scripture Verse Popup Preview ESV (the "Extension") handles your information. Your privacy is a top priority.

1. Information We Do Not Collect

The Extension is designed to function without collecting or transmitting any of your personal or sensitive user data. Specifically, we do not collect, store, track, or transmit:

  • Your browsing history or the content of websites you visit (other than the Bible references you select).
  • Your IP address or any other network identifiers.
  • Any personal information such as your name, email address, or location.
  • Any usage analytics or telemetry about how you interact with the Extension, beyond what is aggregated anonymously by the Chrome Web Store itself for all extensions.

2. Information Stored Locally

To enhance your experience and provide configurable options, the Extension stores certain preferences locally in your browser using Chrome's secure storage (`chrome.storage.sync`). This data is synchronized across your signed-in Chrome browsers but is not accessible to the Extension author or any third party, other than Google as part of its Chrome Sync service. This locally stored data includes:

  • ESV API Key: If you choose to use the ESV translation and provide an API key in the Extension's options page, this key is stored locally. It is only sent directly to the ESV API (https://api.esv.org/) when you request verses from the ESV translation.
  • Default Translation Preference: The default Bible translation you select in the options page.
  • Last Selected Translation: The last translation you chose in the verse preview popup, to remember your preference for the next popup.

This data remains on your computer and is managed by your browser.

3. Communication with External Bible APIs

When you select a Bible reference or use the browser action popup to look up a verse, the Extension communicates with external, public Bible APIs to retrieve the scripture text. The specific API used depends on the translation selected, as configured in the Extension's `bibles.json` file. These APIs may include:

  • ESV API (api.esv.org): When ESV translation is used. Your ESV API key (if provided via `bibles.json` through the build process, or potentially from local storage if using older versions or local modifications) and the requested Bible reference are sent.
  • CDN Bible API (cdn.jsdelivr.net/gh/wldeh/bible-api/): For translations like KJV, ASV, WEB. Only the requested Bible reference (and necessary path components for the API) are sent. No user-specific API key is sent from the extension for these.
  • Bible API (bible-api.com): Can also be used for various translations. Only the requested Bible reference is sent. No user-specific API key is sent from the extension for these.
  • American Bible Society API (api.scripture.api.bible): If configured for translations like NKJV or NASB. An API key (from `bibles.json`, ideally injected during build) and the requested reference/Bible ID are sent.

The Extension only sends the minimal information required to fetch the requested scripture (i.e., the Bible reference and API key where necessary as per `bibles.json` configuration). No user-identifiable information beyond what is intrinsic to an API request (like your IP address, which is handled by the API provider according to their own privacy policies) is sent by this Extension to these APIs.

We encourage you to review the privacy policies of these third-party Bible API providers if you have concerns about how they handle data.

4. `bibles.json` Configuration File

The Extension includes a `bibles.json` file that maps Bible translation codes to their display names and API configurations. This file is part of the Extension's static assets and is fetched locally using `chrome.runtime.getURL('bibles.json')`. API keys for services like ESV or scripture.api.bible are intended to be placeholders in the repository version of this file and replaced with actual keys during a build process (e.g., via GitHub Actions secrets) for published versions. For KJV, ASV, WEB, etc., that use keyless public APIs (like the CDN or bible-api.com), no keys are stored or needed in this file. No external communication is initiated by the Extension to fetch or update this configuration file post-installation.

5. Security

We take reasonable precautions to protect the information stored locally by the Extension. However, no method of electronic storage is 100% secure. Your ESV API key is stored as you provide it, and it's your responsibility to ensure its security if obtained from Crossway.

6. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy within the Extension or on its Chrome Web Store listing. You are advised to review this Privacy Policy periodically for any changes.

7. Contact Us

If you have any questions or concerns about this Privacy Policy or the Extension's practices, please contact the extension author via the Chrome Web Store listing for Scripture Verse Popup Preview ESV.
