Privacy Policy

Scripture Verse Popup Preview ESV is designed with your privacy in mind. We do not collect, store, or transmit any personal, sensitive, or identifiable user data.

  • No personal information is collected or shared.
  • No browsing history or website content is tracked or stored.
  • All user preferences (such as translation choice or API key) are stored locally in your browser using Chrome's secure storage and are never transmitted to any server controlled by the extension author.
  • The extension only communicates with public Bible APIs to retrieve verse text when you request it. No user-identifiable information is sent in these requests.

If you have any questions or concerns about privacy, please contact the extension author via the Chrome Web Store listing.
